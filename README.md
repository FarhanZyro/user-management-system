<h1>User Management System</h1>

<p>A simple PHP-based system for user registration, login, and dashboard access with role-based control.</p>

<h2>Features</h2>
<ul>
  <li>User Registration & Login</li>
  <li>Password Hashing</li>
  <li>Session Management</li>
  <li>Role-based Access (Admin & User)</li>
</ul>

<h2>Roles</h2>
<ul>
  <li><b>Admin:</b> Can access all system features</li>
  <li><b>User:</b> Limited access to dashboard</li>
</ul>

<h2>Technologies</h2>
<ul>
  <li>PHP</li>
  <li>MySQL</li>
  <li>HTML</li>
</ul>

<h2>How to Run</h2>
<ol>
  <li>Place project in htdocs</li>
  <li>Start Apache & MySQL</li>
  <li>Open: http://localhost/user-management/</li>
</ol>
